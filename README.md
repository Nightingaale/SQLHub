## Technology  

- **Java(17)**
- **Gradle**
- **Hibernate**
- **PostgreSQL**
- **Docker**
  
## Information for the repository:

1. **Clone the repository**
``` bash  
git clone https://github.com/Nightingaale/SQLHub.git
```

2. **Find directory**
``` bash
cd ~/SQLHub
```

3. **Build the project**
``` bash
./gradlew build
```

## Information for Docker
``` bash
docker compose up -d
```
